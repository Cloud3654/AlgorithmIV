Node 변수형 생성
	cld(pair-first는 자신의 번호, pair-second는 가는 데 드는 가중치) 생성
	visit(방문 체크) 생성
nd(트리) 생성
n(노드 수), S(시작 번호), D(목적지 번호), minPath(최소 가중치) 생성, S, D 정의
dfs(가중치 탐색 함수) 정의
	현재 인덱스가 목적지라면
	minPath를 가중치의 합과 최소 가중치의 최솟값으로 변경
	함수 종료
idx의 모든 자식 노드에 대해 반복
	방문한 적이 없다면
		방문 체크
		가중치에 방금 이동한 가중치를 더한 수를 2번째 인수로 dfs 호출
main 코드
	m(간선 수) 생성
	n, m 입력
	sp(시작 지점), ep(다음 지점), w(가중치) 생성, m번 반복
		sp, ep, w 입력
		sp의 자식 목록에 {ep, w} 추가
	1번 노드 방문 체크
	시작 지점에서 누적 가중치 0으로 dfs 호출
	minPath 출력
